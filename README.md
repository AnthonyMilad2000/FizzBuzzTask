# FizzBuzz Task

This is a C# implementation of a FizzBuzz coding challenge.

## Description
The application reads input text, splits it into words manually (without using Split or Regex), and applies FizzBuzz rules based on word positions.

- Every 3rd word → Fizz
- Every 5th word → Buzz
- Every 15th word → FizzBuzz

## Features
- Manual string parsing
- OOP design
- Console input/output
- Unit tests for major scenarios

## Tech
- C#
- .NET 9

## How to run
1. Build the solution
2. Run the console application
3. Enter number of lines and input text
